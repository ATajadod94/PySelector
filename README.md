# PySelector
GUI to select reaches and reach samples based on trajectory and velocity profile.

PySelector is developed on Python 3.6. 

Install dependencies:

* cycler==0.10.0
* kiwisolver==1.0.1
* matplotlib==2.2.2
* numpy==1.14.5
* pandas==0.23.3
* pathlib==1.0.1
* pyparsing==2.2.0
* Pypubsub==4.0.0
* python-dateutil==2.7.3
* pytz==2018.5
* scipy==1.1.0
* six==1.11.0
* wxPython==4.0.3
